# goit-algo-hw-02

Домашнє завдання до теми 2 «Основні структури даних».

## Структура

- `task_1.py` — імітація черги заявок у сервісному центрі (`queue.Queue`).
- `task_2.py` — перевірка рядка на паліндром через двосторонню чергу (`collections.deque`).

## Запуск

Потрібен Python 3.10+ (через `X | None`-синтаксис анотацій).

### Завдання 1

```bash
# Інтерактивне меню (за замовчуванням):
python task_1.py

# Автоматичний режим — імітує потік заявок із випадковою генерацією/обробкою:
python task_1.py --auto
```

Меню:
1. Створити нову заявку — викликає `generate_request()`.
2. Обробити заявку з черги — викликає `process_request()`.
3. Показати розмір черги.
0. Вийти.

### Завдання 2

```bash
python task_2.py
```

Програма спочатку проганяє таблицю тест-кейсів (парна/непарна довжина,
регістр, пробіли, фрази), потім переходить у інтерактивний режим.
